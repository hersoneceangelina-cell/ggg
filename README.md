# Receptes Grāmata

## Apraksts
Receptes Grāmata ir vienkārša aplikācija, kas palīdz lietotājam saglabāt, organizēt un atrast savas iecienītākās receptes.  
Lietotājs var pievienot jaunas receptes, meklēt pēc nosaukuma vai sastāvdaļām, kā arī saglabāt receptes sarakstā nākamai gatavošanai.  

---

## Uzstādīšana

1. Lejupielādē projektu:
   ```bash
   git clone https://github.com/username/receptes-gramata# ggg



2.Atver projekta mapi:
cd receptes-gramata

3.Instalē nepieciešamās bibliotēkas:
npm install

4.Palaid aplikāciju:
npm start

## Lietošana
Atver aplikāciju
Nospied "Pievienot recepti"
Ievadi receptes nosaukumu, sastāvdaļas un sagatavošanas aprakstu
Saglabā recepti

Receptes tiks saglabātas sarakstā un būs viegli pieejamas nākamajai lietošanai.

## Funkcijas
Receptes pievienošana
Receptes dzēšana
Receptes meklēšana pēc nosaukuma vai sastāvdaļām
Receptes saglabāšana sarakstā
Receptes rediģēšana
Kā pievienot recepti
Nospied pogu "Pievienot recepti"
Ievadi receptes nosaukumu (piemēram: "Cepts lasis ar dārzeņiem")
Ievadi sastāvdaļas (piemēram: "lasis, burkāni, olīveļļa")
Ievadi gatavošanas aprakstu
Nospied "Saglabāt"

Receptes parādīsies galvenajā sarakstā un būs pieejamas nākamai lietošanai.

# Dzīvnieku Adopcijas Aplikācija

## Apraksts
Dzīvnieku Adopcijas Aplikācija palīdz lietotājiem atrast un adoptēt mājdzīvniekus no tuvākajām patversmēm.  
Lietotājs var meklēt dzīvniekus pēc veida, vecuma vai atrašanās vietas un pieteikties adopcijai.

---

## Uzstādīšana

1. Lejupielādē projektu:
   ```bash
   git clone https://github.com/username/dzivnieku-adopcija

2. Atver projekta mapi:
cd dzivnieku-adopcija

3. Instalē nepieciešamās bibliotēkas:
   npm install

4. Palaid programmu:
npm start

## Lietošana
Atver aplikāciju
Izvēlies dzīvnieku veidu un atrašanās vietu
Skati pieejamos dzīvniekus
Nospied "Pieteikties adopcijai"
## Funkcijas
Dzīvnieku meklēšana pēc veida un vecuma
Dzīvnieku profilu apskate
Adopcijas pieteikuma iesniegšana
Saglabā iecienītākos dzīvniekus
Filtrēšana pēc atrašanās vietas
Kā pieteikties adopcijai
Atver dzīvnieka profilu
Nospied pogu "Pieteikties adopcijai"
Ievadi savus kontaktinformāciju
Nospied "Nosūtīt pieteikumu"

Dzīvnieka adopcijas pieteikums tiks nosūtīts patversmei.
